WorkSpace Manager — Privacy Policy

Last updated: April 2026

Data Collected

This extension reads the URLs and titles of open Chrome tabs solely when the user explicitly triggers a session save.

How It Is Used

Tab URLs and titles are sent to the WorkSpace Manager desktop application running locally on the same machine. They are stored in a local file on the user's device for the purpose of restoring the session later.

Data Sharing

No data is transmitted to any remote server, third party, or external service of any kind.

Data Retention

Data is stored locally and can be deleted at any time by removing the saved session from the desktop app.
